# Changelog

## v1.2.1 — 2026-03-23

- Fixed audio track grouping and dropped bitrate from track labels in `--list-audio-tracks` output.

- Initial public release.

## v0.1.0 — 2026-03-23

- Initial commit: CLI tool for generating subtitles from video files using Whisper.
